## Overview

This project showcases an interactive Sales Performance Dashboard built using Microsoft Excel. The dashboard analyzes sales data across multiple cities and sales executives, providing insights into total sales, target achievement, and performance gaps through dynamic visualizations.

### Project Objectives

Analyze sales performance by Sales Executive

Track Total Sales across different cities

Measure Target Hit Percentage

Identify gap from sales targets

Enable dynamic city-wise filtering
